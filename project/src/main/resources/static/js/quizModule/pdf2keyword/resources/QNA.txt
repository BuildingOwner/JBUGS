Q. Linked List 자료 구조에서 head는 무엇을 의미하는가?
A. Head는 Linked List의 첫 노드를 참조하는 포인터를 의미한다.
Q. 다음 코드가 가리키고 있는 링크는 무엇인가?
Node *head;
A. head 노드가 참조하고 있는 링크를 가리키고 있다.
Q. 다음의 코드를 실행하면 출력되는 값은?
String name = null; 
System.out.println(name);
A. null
Q. dlistnode구조체에 head가 있는지 여부를 판단하는 방법은 무엇입니까?
A. dlistnode구조체에서 prev 포인터가 생략됐는지 확인하는 것이 방법입니다. head가 있다면 prev 포인터는 무조건 NULL 상태여야합니다.
Q) 리스트의 요소를 저장하기 위해서는 어떤 자료구조를 사용하는가?
A) 연결 리스트를 사용하여 리스트의 요소를 저장합니다.
Q. 다음 코드에서 변수 node가 가지고 있는 속성은 무엇인가?
struct Node 
{ 
   int data; 
   struct Node* next; 
}; 
A. 변수 node가 가지고 있는 속성은 int data, 그리고 가리키는 address인 struct Node* next입니다.
Q. rlink는 다음과 같이 두 개의 노드를 연결하기 위해 사용될 수 있는 자료구조를 말하는 것인가?
A. 예, 맞습니다. rlink는 두 개의 노드를 간단하게 연결하기 위해 사용되는 자료구조입니다.
